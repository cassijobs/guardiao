# Guardião v2.0.1

Versão corrigida e funcional do Guardião.

## Como testar

Abra `index.html` no navegador ou publique a pasta no GitHub Pages.

## Reiniciar o primeiro encontro

Use:

`index.html?reset=1`

ou, no GitHub Pages:

`https://seuusuario.github.io/seurepositorio/?reset=1`

## Estrutura

- `index.html` - estrutura da página
- `style.css` - visual
- `js/config.js` - configurações
- `js/memoria.js` - memória local
- `js/palco.js` - interface
- `js/condutor.js` - interpreta cenas
- `js/script.js` - inicia o encontro
- `encontros/primeiro.js` - primeiro encontro
- `encontros/segundo.js` - segundo encontro
- `encontros/terceiro.js` - terceiro encontro
